## 📌 Project Overview

The **Padawigama Village Portal** is a web-based management system built for the two village societies of Padawigama village, Sri Lanka. It allows members, office bearers, and society presidents to manage all society-related activities through a single digital platform.

---

## 🏘️ About the Societies

### ⚰️ Eksath Maranadara Samithiya
A **Funeral Assistance Society** that provides financial aid and manpower to member families during bereavement.
- Lump sum of **Rs. 100,000** per death event
- Members pay **Rs. 1,000/month**
- Provides manpower for all funeral tasks

### 🌾 Ranketh 314/D03 Govisamvidana Samithiya
A **Farmer Organization** managing water distribution from canal 314/D3 to all paddy fields.
- 90 farmer members, 90 hectares
- Field canals FC 5 to FC 12
- Organizes regular canal cleaning

---

## ✨ Features

### 🔐 Authentication
- Member login via NIC number + password
- New member self-registration
- Password change
- President-only protected edit access

### ⚰️ Maranadara
- Member registry with search and filters
- Individual member profiles with 12-month payment history
- Office bearers management
- Monthly payment tracking
- 3-day funeral duty roster
- Defaulter tracking (6+ missed payments)

### 🌾 Govisamvidana
- Farmer registry with paddy field details
- Canal map SVG + real canal board photo
- Monthly water distribution schedule
- Canal cleaning schedule

### 🔑 President Edit Access (password protected)
- Add / Remove members
- Edit office bearers and committee
- Record payments
- Edit funeral duty assignments
- Edit water and cleaning schedules

---

## 🛠️ Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Bootstrap 5.3 (mobile responsive)
- Bootstrap Icons
- Google Fonts (Playfair Display, DM Sans, DM Mono)
- No backend, no database, no build tools required
